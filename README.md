# LearnML
LearnML은 머신러닝 학습자를 위해 개발된 시각화 프로그램으로, Windows만 지원합니다.

## 특징
- LearnML 확장을 통해 기능을 무한 확장할 수 있습니다. LearnML 확장 개발에 관한 정보는 [이곳](https://github.com/kmc7468/LearnML/wiki/%ED%99%95%EC%9E%A5-%EA%B0%9C%EB%B0%9C)을 참조하세요.
